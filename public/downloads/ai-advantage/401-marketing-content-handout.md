# AI Advantage 401 — Marketing and Content Handout

One page. Publish 3 pieces a week, compliance checked.

---

## The 3 Pieces a Week Rule

Not 10. Three. One GBP post, one short video script, one social post. Run each through your Twin, edit, run through the safe content decision tree, publish.

## The Safe Content Decision Tree

Before any post goes public, ask:

1. Does it mention a specific rate, payment, or fee? → Full disclosures required. Send to compliance.
2. Does it name a competitor? → Rewrite. No competitor comparisons.
3. Does it imply a guaranteed outcome? → Rewrite. No "guaranteed approval" or "lowest rate."
4. Does it include your NMLS ID and Equal Housing logo (where required)? → Add if not.
5. Could a reasonable borrower read it as advice for their specific situation? → Soften. Add "talk to a licensed loan officer."

If any answer is "yes" but you didn't fix it, **do not publish**.

## Pick One Pillar

Not three. One borrower profile or topic for the next 90 days:

- First time buyers
- Self employed
- VA / military
- Investor
- Move up
- Divorce
- New construction
- Reverse

Everything you post ties back to that one pillar.

## The Three Templates

### Google Business Profile Post

```
[Market stat or local fact]. [One sentence about what it means for 
borrowers in your area]. [One sentence about how to start a 
conversation]. NMLS [ID].
```

### Short Video Script (60 sec)

```
[0-5 sec] Hook. Pattern interrupt. No intro.
[5-30 sec] One example. One story. One data point.
[30-55 sec] The takeaway. What to do, what to avoid.
[55-60 sec] CTA. "DM me 'yes' if you want the next video."
```

### Social Post

```
[Hook in first line — visible before "see more"].
[2-3 short lines. The pattern interrupt or the data point.]
[NMLS ID]
[Plain CTA: link in bio, DM me, or call].
```

## The Multi-Language Path

If your market has bilingual borrowers, draft both versions in your Twin, then have a native speaker review before posting. The Twin is the first draft, not the final.
