## The Plan, In One Page

```
Week 1-2:  Foundation. Twin is built. 3 sources connected. Bio doc 
           uploaded. First 3 drafts sent.

Week 3-4:  Daily loop. 5 first-touch messages/week drafted by the 
           Twin, edited, sent. Status updates live.

Week 5-6:  Content cadence. 3 pieces/week published. Safe content 
           decision tree run on every piece.

Week 7-8:  First automation. Reactivation sequence live. Status 
           update template live. Documented.

Week 9-10: Mid review. Audit the 3 workflows. Cut what is not 
           working. Double what is. Recommit or pivot.

Week 11-12:Final push. Measure the lift. Submit your 12 week plan 
           and your 3 workflows with success metrics.
```

## The 3 Workflows That Compound

Pick 3 from this list (or your own). The rule: each must save you 30+ minutes/week OR generate 1+ new conversation/week.

| Workflow | What it does | Time saved / lift |
| --- | --- | --- |
| Reactivation sequence | Past client touch | X replies per month |
| Status update template | Friday borrower email | Zero confusion calls |
| GBP weekly post | One post every Friday | X profile views / calls |
| Short video script | One script per week | X views / DMs |
| Social post | One post per week | X engagement |
| Past client check in | Quarterly call script | X closed referrals |

## Success Metrics (Pick Per Workflow)

For each workflow, define one metric. The metric must be:

- **Countable.** Number, not vibe.
- **Owned by you.** Not "borrowers should reply." But "I send 5 per week."
- **Time bounded.** Weekly or monthly check.

```
Workflow: Reactivation sequence
Metric: 10 past clients touched per week
Owner:  Me
Review: Every Monday at 9am
```

## Quarterly Review

Every 90 days, sit down with your coach or team leader and walk through:

1. Which workflows are still running?
2. Which workflows delivered the metric?
3. Which workflows are dead? Cut them.
4. What is the new workflow for the next 90 days?
