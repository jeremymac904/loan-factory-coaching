## How To Export PDF Manually

From Chrome:

1. Open the HTML file.
2. Press `Cmd+P`.
3. Destination: Save as PDF.
4. Layout: Landscape.
5. Paper size: 16:9 if available, otherwise use default landscape.
6. Background graphics: On.
7. Save.

The print stylesheet hides presenter notes and exports one clean 16:9 slide per page.
